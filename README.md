* 만든 목적 

1. 평소에 주식에 관심을 가지고 있고 파이썬을 깊게 공부 할겸 관심을 가는 자동프로그램의 인강을 수강하게 되었습니다.

2. 취미생활 또는 재테크로 돈벌기를 하고 싶어 수강하게 되었습니다.

* 추후계획

1. 인강을 다 듣고 나서 나만의 주식전략 알고리즘을 만들어 소스를 변경할 예정입니다.

2. 꾸준히 투자 봇을 공부해 알고리즘 및 소스를 수정할 계획입니다.
